# Overmotivator Front End

## Requirements

* npm install -g @angular/cli

## Install

```
npm install
```

## Configure

Create file config.json like this one:
```
{
  "appId": "backedAppId",
  "serverUrl": "http://localhost:1337/parse"
}
```
or reuse one coming from https://github.com/bjakubiak/overmotivator-be

## Start

For development:
```
ng serve
```
